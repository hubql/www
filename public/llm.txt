## Technology Stack

### Frontend & Design
- **Figma**: Collaborative design, prototyping, component libraries, design systems
- **React**: Component-based UI architecture, modern patterns, Next.js integration
- **Next.js**: Server-side rendering, static generation, App/Pages Router
- **Tailwind CSS**: Utility-first styling, responsive design, custom design systems
- **Framer Motion**: Animations and interactive UI elements
- **TypeScript**: Type-safe development across the stack

### Backend & Infrastructure
- **Node.js**: Backend runtime for scalable server-side applications
- **Supabase**: PostgreSQL database, authentication, real-time subscriptions, storage
- **AWS**: Cloud hosting, serverless architecture, managed services, CloudFront CDN
- **Cloudflare**: Global CDN, DDoS protection, edge optimization, Workers deployment

### Automation & Integration
- **n8n**: Workflow automation, API orchestration, self-hosted integrations
- **Zapier**: No-code integrations and automated workflows

### Development & Deployment
- **Git & GitHub**: Version control and collaboration
- **TinaCMS**: Content management system
- **OpenNext for Cloudflare**: Next.js deployment to Cloudflare Workers
- **PostHog & Koala**: Analytics and user tracking
